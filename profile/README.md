# Development playground

Hey there, this is the dev playground owned by [@giftkugel](https://github.com/giftkugel). :wave:

It contains open sources projects in Java, Go and Web technologies.

## Featured projects

- [STOPnik](https://github.com/webishdev/stopnik) an implementation of OAuth and OIDC, see https://stopnik.webish.dev/
- [fail2ban-dashboard](https://github.com/webishdev/fail2ban-dashboard) a simple dashboard for [fail2ban](https://github.com/fail2ban/fail2ban)
- [password](https://github.com/webishdev/password) yet another password generator without cookie nor server communication, see https://password.webish.dev/
